Webservers

Webservers deffinition

is software that serves web pages to users over the Internet. When a user enters a URL in their browser, the web server processes the request and delivers the requested content, such as HTML pages, images, or files.


Most popular types of webservers

  • Apachi Webserver
  • Nginx Webserver

what is the difference between Apachi & Nginx webserver?

apachi webserver:
Due to its process-based model, Apache can consume a significant amount of memory and CPU resources when handling multiple connections. Each request requires its own process or thread, leading to overhead. Under high load, this can result in slower performance and potential server bottlenecks.
Nginx webserver
Nginx’s event-driven architecture allows it to handle many concurrent connections with minimal resource usage. It can serve static files and handle requests without creating new processes for each connection. This leads to better performance and responsiveness, especially under high traffic conditions.
